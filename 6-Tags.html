Tags para deixar o texto semântico:
Normalmente é possível que as tags do tipo "bloco" possuam outras tags dentro delas (tags filhas) O bloco usa a largura inteira do elemento para quebrar linhas
Tags para texto (p): b e strong deixam o texto em negrito, porém strong significa deixar a palavra mais forte para ser lida.
Strong é um estado lógico e Bold é um estado fisico.
A tag i serve para escrever em italico e a tag emfase (em) também deixa o texto em italico porém é para dar emfase em uma palavra no estado lógico.
O estado lógico é o momento em que o leitor de texto lê a tela do dispositivo.

A tag del serve para escrever o texto tachado. A tag ins é insert e faz o sublinhado na palavra.
O sentido é que del é para deletar um texto e ins é para inserir um texto.

A tag sup serve para fazer um número elevado, por exemplo 2^2 = 22 ou H2O. A tag q coloca o texto entre aspas ou para fazer a citação de um texto. Exemplo: H2O

O blockquote serve para fazer um bloco de citação.
O paragrafo não pode ter um bloco, mas o bloco pode ter um paragrafo
